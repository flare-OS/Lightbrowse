Introducing LightBrowse

A fast, lightweight browser

LightBrowse is a no-nonsense web browser designed for older Windows systems. Clean interface, low memory usage, and no telemetry. Browse the modern web on vintage hardware.

About LightBrowse

About the Project
Work In Progress

LightBrowse is an independent open-source web browser project focused on delivering a usable, modern browsing experience on older Windows versions.

The project is currently in early development. A public alpha is planned for release once core rendering and navigation functionality is stable.

Built with love for anyone who still runs Windows XP through 7 and wants a browser that doesn't phone home.
Planned Features
Roadmap
  • Tabbed browsing with tab groups
  • Custom user agent switching
  • Built-in ad/tracker blocker
  • Download manager with resume
  • Bookmark sync (local, no cloud)
  • Extension/plugin support
  • Session restore on crash
  • Custom themes & skins
System Requirements
All Platforms
  • OS: Windows XP SP3 or later
  • CPU: 800 MHz or faster
  • RAM: 256 MB minimum
  • Disk: 30 MB free space
  • Display: 800x600 or higher

Also works great on Windows Vista, 7, 8, 8.1, and 10. Linux/Wine support is planned.
